현재 비디오 생성 AI 시장은 춘추전국시대입니다. 각 서비스마다 강점이 다르므로, 목적(실사, 애니메이션, 광고 등)에 맞는 도구를 선택하는 것이 중요합니다.

## 📊 주요 서비스 상세 비교

| 서비스 | 주요 강점 | 무료 플랜 | 유료 시작 가격 | 영상 길이 |
| :--- | :--- | :--- | :--- | :--- |
| **Luma Dream Machine** | 사실적인 물리 엔진, 높은 실사 퀄리티 | 월 30회 생성 | $29.99 / 월 | 기본 5초 |
| **Runway Gen-3 Alpha** | 정교한 카메라 컨트롤, 업계 표준 기능 | 맛보기 크레딧 제공 | $12 / 월 (Standard) | 5~10초 |
| **Kling AI** | 최대 2분 생성 가능, 압도적인 일사 표현 | 매일 무료 크레딧 | 약 $10 / 월 | 5초 ~ 2분 |
| **Pika Art** | 사운드 생성(SFX), 립싱크, 스타일 변환 | 매일 크레딧 리필 | $8 / 월 (Starter) | 3~4초 |

## 🔍 서비스별 심층 분석

### 1. Luma Dream Machine (최고의 물리 체감)
- **장점**: 
    - 사물과 인간의 움직임이 매우 자연스럽고 물리 법칙을 잘 따릅니다.
    - **Image-to-Video** 성능이 현재 업계 최상위권입니다.
- **단점**: 
    - 무료 사용자가 많아 대기 시간이 길어질 수 있습니다.
    - 영상 길이가 5초 단위로 다소 짧습니다.
- **비용**: 개인 사용 시 월 30회 무료 제공 후, $30(약 4만원)부터 시작하는 유료 플랜이 있습니다.

### 2. Runway Gen-3 Alpha (전문가용 도구)
- **장점**: 
    - **카메라 모션 브러시**: 특정 부분만 움직이게 하거나 카메라 이동 방향을 세밀하게 조정 가능합니다.
    - 일관성이 매우 높아 상업 광고 제작에 가장 많이 쓰입니다.
- **단점**: 
    - 사실상 무료 플랜은 거의 체험 수준이며, 제대로 쓰려면 유료 결제가 필수입니다.
- **비용**: 연간 결제 시 월 $12부터 시작하며, 무제한 생성(Relaxed 모드)은 $76 플랜부터 지원합니다.

### 3. Kling AI (가장 긴 생성 시간)
- **장점**: 
    - 기존 5~10초의 한계를 깨고 **최대 2분**까지 영상을 이어 붙일 수 있습니다.
    - 사람의 얼굴 표현과 복잡한 동작(식사하기 등) 묘사가 매우 정교합니다.
- **단점**: 
    - 중국 기반 서비스로 시작되어 글로벌 결제나 가입 절차가 다소 번거로울 수 있습니다 (현재 글로벌 버전 출시됨).
- **비용**: 가입 시 매일 66크레딧(약 6회 생성 가능)을 무료로 주며, 유료 플랜은 약 $10 내외입니다.

### 4. Pika Art (창의적이고 재미있는 기능)
- **장점**: 
    - **Sound FX**: 영상에 맞는 효과음을 자동으로 생성해 줍니다.
    - **Lip Sync**: 인물의 입모양을 목소리에 맞게 맞출 수 있습니다.
- **단점**: 
    - 실사 퀄리티(Realism) 면에서는 Luma나 Runway에 비해 약간 떨어질 수 있습니다.
- **비용**: 매일 30크레딧씩 리필되어 가볍게 쓰기에 가장 좋습니다.

## 💡 어떤 것을 선택해야 할까?

- **"진짜 사진 같은 고퀄리티 영상을 원한다"** -> **Luma** 혹은 **Runway**
- **"카메라 워킹을 내가 직접 조절하고 싶다"** -> **Runway**
- **"최대한 긴 영상을 만들고 싶다"** -> **Kling**
- **"사운드까지 한꺼번에 해결하고 싶다"** -> **Pika**
